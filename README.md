## Project Setup

### Prepare the Docker template

1. Clone this repo:

```
git clone https://github.com/LibardoZuniga/lunch_donation.git
cd lunch_donation

```
2. Initialize Docker:

```
cd docker
docker-compose build
docker-compose up -d
```
